Utilities

CalibrateEmulateSample.MarkovChainMonteCarlo.MCMC — Type
MCMC{FT<:AbstractFloat, IT<:Int}

Structure to organize MCMC parameters and data

Fields

  • obs_sample

    a single sample from the observations. Can e.g. be picked from an Obs struct using getobssample

  • obs_noise_cov

    covariance of the observational noise

  • prior

    array of length N_parameters with the parameters' prior distributions

  • step

    MCMC step size

  • burnin

    Number of MCMC steps that are considered burnin

  • param

    the current parameters

  • posterior

    Array of accepted MCMC parameter samples. The histogram of these samples gives an approximation of the posterior distribution of the parameters. paramdim x nsamples

  • log_posterior

    the (current) value of the logarithm of the posterior (= loglikelihood + logprior of the current parameters)

  • iter

    iteration/step of the MCMC

  • accept

    number of accepted proposals

  • algtype

    MCMC algorithm to use - currently implemented: 'rmw' (random walk Metropolis)
